Location and Position¶

These functions usually are found the the preamble or the epilog of your application code using the CGNS/MLL.

Grid Location

cg_gridlocation_write

Write grid location

cg_gridlocation_read

Read grid location

Point Sets

cg_ptset_write

Write point set data

cg_ptset_info

Get point set information

cg_ptset_read

Read point set data

Rind Layers

cg_rind_write

Write number of rind layers

cg_rind_read

Read number of rind layers

Grid Location¶

Node: GridLocation_t (SIDS, File Mapping)

Point Sets¶

Node: IndexArray_t, IndexRange_t (SIDS, File Mapping)

Rind Layers¶

Node: Rind_t (SIDS, File Mapping)
